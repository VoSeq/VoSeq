Create taxonset¶

Taxonset is a way to make a list of taxa that are being used for a specific project or analysis. A Taxonset is just a list of voucher codes. By having Taxonsets, you can quickly create datasets and tables for them.

If you have not set Taxonsets you will need to type specimen codes everytime you create a dataset. Instead, if you have a Taxonset for a particular project, you could select it when creating Tables for manuscripts.

Go to the Administrator interface and click on the link Biological data » Taxon sets » Add taxon sets.

A taxonset must have a name in order to be saved and usable later!
