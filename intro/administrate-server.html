Administrate the server with your installation of VoSeq¶

Create an administrator account for your VoSeq installation¶

Before you can start adding content to VoSeq you need to create an account for the administrator. This administrator then will be able to log into VoSeq and create user accounts for collaborators with several degrees of privileges.

Open a terminal and type the following commands:

cd /path/to/VoSeq
make admin

You will be asked for username, password and email accounts.
